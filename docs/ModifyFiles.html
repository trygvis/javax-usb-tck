Modify Files for Building Images


The Cypress EZ-USB FX Development Kit provides several C Language Files that are built into the EEPROM Images for the Development Boards.

These files must be personalized to operate correctly with the JSR80 javax.USB TCK. Please follow the instructions below as to which files will be changed and how those files need to be modified. These modifications will cover everything necessary for all three images used in the JSR80 javax.USB TCK tests (BULKINT.B6, ISODCP.B6 and TOPOLOGY.B6).

Note: The following instructions are based on the EZ-USB Development Tools v2.61.700. If you are using a different version of the EZ-USB Development Tools, the files may look somewhat different. Please make the modifications below accordingly.

Steps to Modify Files:

  1. Modify the FW.C File for all three Images: FW.C Modifications
  2. Modify the PERIPH.C File for the BULKINT Image: BULKINT PERIPH.C Modifications
  3. Modify the PERIPH.C File for the ISODCP Image: ISODCP PERIPH.C Modifications

Note: Please be sure that the modified FW.C file has been copied into all three subdirectories (BULKINT, ISODCP, TOPOLOGY).
