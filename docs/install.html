Installation


These instructions assume that all files of the javax.usb TCK are in the directory /javaxusbtck-1_0/.

Build the javaxusbtck.jar file by running "ant jars" from the /javaxusbtck-1_0 directory, note that implementation's and JUnit jars must be in the classpath. To see other ant options, run "ant" from the same directory.

Follow any instructions required to build and install the javax.usb implementation.

To run the javax.usb TCK, your classpath must contain the following:

  • /javaxusbtck-1_0/lib/javaxusbtck.jar
  • junit.jar from JUnit.org
  • the jar file(s) containing the javax.usb implementation to be tested
