Introduction

The javax.usb TCK is the technology compatibility kit for the Java Specification Request JSR80: Java USB API. This Java Specification Request (JSR) is being developed under the Java Community Process.

This TCK provides a means to verify the compliance of an implementation of the JSR80: Java USB API with the technology specification. The TCK does not verify compliance with any other software or hardware technology specification and is not a substitute for other testing such as performance, system, or functional testing of the implementation.

This work is being made available under the IBM Common Public License.

Requirements

The Java USB API provides a means for JAVA programs to communicate with USB devices. In order to verify the API, USB devices are required to represent known configurations of USB devices and verifiable USB functions.

The javax.usb TCK has the following requirements:

Hardware

  • Computer with two or more USB Host Controllers configured to run the javax.usb implementation.
  • 4 - USB(1.1 or 2.0) hubs of any configuration (at least one of these hubs must be USB 1.1)
  • 1 - 4 port USB (1.1 or 2.0) hub
  • Cypress EZ-USB FX Integrated Circuit Xcelerator Development Kit CY3671
    • Computer with at least one available USB port running a Windows version as required by the EZ-USB Development Tools that come with the Development Kit.
  • A low speed USB device
  • A USB device with at least one non-default control pipe

Software

  • JUnit 3.8.1 or later.
  • Ant 1.6.0 or later.
  • Java 1.3.1 or later.
