Compiling EEPROM Images


In order to run the JSR80 javax.USB TCK Tests, the EEPROMs of the Cypress Development Boards must be programmed with the appropriate images.

The following is an explanation of how to build the three images supported by the JSR80 javax.USB TCK: BULKINT.B6, ISODCP.B6, TOPOLOGY.B6.

Note: These are the images that will be used in the next step, Step 4: "Program the Cypress EZ-USB FX Development board with an Image"

Steps to Build the BULKINT.B6 Image:

  1. Open a DOS Command Prompt Window.
  2. Navigate to the BULKINT Image subdirectory, C:/TCK/BULKINT, which was created following completion of step 1 of the "Steps to Build EEPROM Images."
  3. Type the following at the command prompt: bulkint.
  4. To check the success of the compilation, please review the output generated on the command window. If any of the compilation lines indicate that one or more Errors were generated while building a file, the entire build of that Image failed. Please address the problem with the file that did not compile cleanly and repeat this process.
