Building the Images


In order to set up the Cypress development board to run the TCK tests, they must be programmed with the appropriate images.

There are three images used in the JSR80 javax.USB TCK tests: BULKINT, ISODCP, TOPOLOGY. The following instructions will explain the steps necessary to build these three images.

The following instructions are based on the EZ-USB Development Tools v2.61.700. If you are using a different version of the EZ-USB Development Tools, please refer to the tool documentation for any differences encountered between the instructions and your tool.

Steps to Build EEPROM Images:

  1. Gather the necessary files into the correct subdirectories: Instructions to Gather Files
  2. Modify certain files for use with the JSR80 javax.USB TCK: Instructions to Modify Files
  3. Compile the new EEPROM images: Compiling Images Instructions
  4. Program the Cypress EZ-USB FX Development board with an Image: EEPROM Programming Instructions
