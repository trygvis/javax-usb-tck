Gathering Necessary Files


The following steps explain how to set up your system, in order to be able to build the BULKINT.B6, ISODCP.B6 and TOPOLOGY.B6 images for running the JSR80 javax.USB TCK tests. Please set up the subdirectories and files just as is listed below, or the build files will not be able to locate the necessary files.

Note: The locations listed for the files provided through the installation of the Cypress EZ-USB FX Development Kit are based on a typical installation and a custom installation may change the location of those files on your system.

Steps to Gather the Files:

  1. Create a subdirectory on the root of your system called "TCK".
  2. In the new TCK subdirectory, create the following three subdirectories: "BULKINT", "ISODCP", "TOPOLOGY".
  3. In the new BULKINT subdirectory, create a subdirectory called "SUPPORT".
  4. In the new ISODCP subdirectory, create a subdirectory called "SUPPORT".
  5. In the new TOPOLOGY subdirectory, create a subdirectory called "SUPPORT".
  6. Copy the following files from the locations listed next to them into all three SUPPORT subdirectories, which were created in steps 3, 4 and 5:

          FILE                                 LOCATION

    • HEX2BIX.EXE      ===   c:\Cypress\USB\Bin
    • EZUSB.LIB           ===   c:\Cypress\USB\Target\Lib\Ezusb
    • USBJMPTB.OBJ   ===   c:\Cypress\USB\Target\Lib\Ezusb

  7. Copy the following files from the locations listed next to them into the BULKINT, ISODCP and TOPOLOGY subdirectories, which were created in step 2:

          FILE                                 LOCATION

    • FW.C           ===   get the base from C:\Cypress\USB\Target\Fw\Ezusb
    • PERIPH.C    ===   get the base from C:\Cypress\USB\Target\Fw\Ezusb
    • EZUSB.H     ===   c:\Cypress\USB\Target\Inc
    • EZREGS.H   ===   c:\Cypress\USB\Target\Inc

  8. Copy the following files from the locations listed next to them into the BULKINT subdirectory, which was created in step 2:

          FILE                              LOCATION

    • BULKINT.BAT   ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • ABUILD.BAT      ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.A51          ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.H              ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)

  9. Copy the following files from the locations listed next to them into the ISODCP subdirectory, which was created in step 2:

          FILE                           LOCATION

    • ISODCP.BAT   ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • ABUILD.BAT   ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.A51        ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.H            ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)

  10. Copy the following files from the locations listed next to them into the TOPOLOGY subdirectory, which was created in step 2:

          FILE                                 LOCATION

    • TOPOLOGY.BAT   ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • ABUILD.BAT          ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.A51               ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
    • DSCR.H                   ===   source subdirectory of the extracted archive (e.g. /javaxusbtck-1_0/source)
