Instructions to run the Topology Test1


Program the Cypress EZ-USB FX development board with the topology.b6 image. Refer to EEPROM Programming Instructions.

Setup the hardware as per the following Configuration.

Run the following command from the root directory of the extracted archive : /javaxusbtck-1_0>java javax.usb.tck.CONFIG1_TOPOLOGY_B6_IMAGE

To be considered compliant with the javax.usb API specification, your implementation must pass every test in the suite; that is, there must be 0 failures and 0 errors. The execution and output would appear similar to the following:

javaxusbtck-1_0>java javax.usb.tck.CONFIG1_TOPOLOGY_B6_IMAGE
..
Time: 1.602

OK (2 tests)
